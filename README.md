# BRNV Calculator

BRNV Calculator is a lightweight Chrome Extension (Manifest V3) with a polished popup calculator, a larger dedicated view, and bilingual settings.

## Features

- Apple Calculator inspired layout adapted to a white, red, and gray extension-pack style
- Compact toolbar popup with keyboard support
- Large calculator page for more comfortable use
- Recent calculation history with restore and clear actions
- English and Ukrainian interface support
- Synced preferences with `chrome.storage.sync`
- Local history storage with `chrome.storage.local`

## Notes

- Language options: English and Ukrainian
- History is stored locally in the browser
- Preferences are stored with Chrome sync storage
